# Pantonial.Dev - Sample Project

## Project setup
1. Install node modules
```
npm install
```
2. Dockerize on port `8080` (https://v2.vuejs.org/v2/cookbook/dockerize-vuejs-app.html#Real-World-Example)

### Unit tests
```
npm run test:unit
```

### Integration tests
Note: Use port `8081` for integration testing.
```
npm run test:e2e
```
